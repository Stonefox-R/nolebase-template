## 环境创建指令
```text
conda create -n 环境名字
```
## 查看现有环境

```text
conda info --envs
```
## 使用指定环境
```text
Conda activate 环境名
Source activate 环境名
activate + 环境名
```
## 在指定环境中安装包
```text
pip install
conda install
```
## 查看环境中现有包
```text
pip list
conda list
```
## 退出环境
```text
deactivate 环境名
```
## 删除环境
```text
conda remove -n 环境名 --all
```
